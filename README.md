# Credit Card Fraud Detection using Machine Learning

This project detects fraudulent credit card transactions using machine learning techniques.

## Dataset
Kaggle Credit Card Fraud Detection Dataset

## Project Workflow
1. Data Exploration
2. Outlier Detection
3. Feature Scaling
4. Handling Class Imbalance
5. Model Training
6. Model Evaluation

## Model Used
Logistic Regression

## Evaluation Metrics
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
