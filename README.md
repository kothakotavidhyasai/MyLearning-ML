# MyLearning-ML

A collection of Machine Learning projects built from scratch.

## Projects

### 1. House Price Prediction
- Linear Regression + Gradient Descent from scratch
- PCA from scratch
- Washington State housing dataset (4,600 houses)
- R² = 0.43
- [View project](./House_Price_Prediction)

### 2. Air Quality Prediction
- Linear Regression + Gradient Descent from scratch
- PCA from scratch (13 → 6 components)
- UCI Air Quality dataset (9,357 hourly readings)
- R² = 0.90
- [View project](./Air_Quality_Prediction)
